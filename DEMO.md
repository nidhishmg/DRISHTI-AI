# Reality Gap AI - Phase 3 Demo Script

1. Run `make up`.
2. Open `http://localhost:3000`.

## Scenes
1. **Landing**: Show Heatmap. Explain real-time clusters.
2. **Citizen Demo**: Simulate voice upload.
3. **Cluster Explorer**: Show Pension Delays cluster. Point out Trust metrics.
4. **Causal Graph**: Show link between Failure and Network Outage.
5. **Simulator**: Adjust sliders to show impact on exclusion vs fraud.
6. **Export**: Show PDF preview.
